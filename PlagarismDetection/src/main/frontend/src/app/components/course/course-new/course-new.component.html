Adding a New Course (Read Me)


The GitHub Repo URL refers to the course repository. In order to successfully add a course, a certain GitHub folder structure is required. The assignment folders should be inside the course folder, and in each assignment folder there should be a project folder for each student for that particular assignment.

      |- HW 2
      |- HW 3
